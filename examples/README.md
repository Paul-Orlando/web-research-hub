Token economics and AI infrastructure CapEx is a fast-moving,
genuinely complex topic with no single authoritative source.
This output shows the pipeline successfully:

1. Splitting a broad question into two distinct, well-scoped
   search angles
2. Retrieving real, current external data via Exa rather than
   relying on model training knowledge
3. Synthesizing competing/complementary sources into a coherent,
   categorized report a person could actually use
